# Simple Calculator Web App

## Description
This is a simple web-based calculator built using HTML, CSS, and Java Servlet. 
It performs basic arithmetic operations: Addition, Subtraction, Multiplication, and Division.  

## Features
- Input two numbers
- Select operation (+, -, *, /)
- Display result on the same page
- Handles division by zero
- Styled with CSS, centered layout
- "Go Back" link included for convenience

## Technologies Used
- HTML
- CSS
- Java Servlet
- Apache Tomcat 

## How to Run
1. Clone the repository
2. Deploy on a Java EE server (e.g., Tomcat)
3. Open `Form.html` in browser
4. Enter numbers and select operation
5. Click Calculate to see the result
